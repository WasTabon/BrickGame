# BrickGame — Iteration 17: Battle Buffs (3 случайных на бой)

## 1. Что было добавлено
Перед каждым боем выпадает оверлей с **3 случайными бустами на выбор**. Выбранный буст действует только на этот заход боя и **не сохраняется** — при перезаходе/повторе выпадет новый выбор.

## 2. Пул бустов (аркадно-сильные)
- **DOUBLE DAMAGE** — x2 урон за выстрел.
- **RAPID FIRE** — стрельба вдвое чаще.
- **EXTRA AMMO** — +60% атак.
- **TWIN SHOT** — два выстрела по двум целям за один заряд.
- **PIERCE** — снаряд пробивает врагов насквозь.
- **BIG SHELLS** — взрыв с уроном по площади.
- **SLOW FIELD** — враги движутся вдвое медленнее.

Из 7 случайно показываются 3.

## 3. Что изменилось
- `BattleBuffs` (новый) — пул, выбор `Choose`, `PickThree()`, сброс `Reset()`. Состояние статическое и живёт только на текущий бой.
- `BattleBuffPicker` (новый) — оверлей с 3 картами; по тапу применяет буст и запускает бой.
- `BattleController` — на старте показывает пикер, бой начинается только после выбора; применяет эффекты (урон/скорострельность/патроны/мульти-цель/пробитие/взрыв/замедление).
- `Projectile` — поддержка `pierce` (не уничтожается, бьёт несколько) и `explosive` (AoE по `OverlapCircleAll`).

## 4. Editor скрипты: что запустить
1. Скопируй папку `Assets/BrickGame` поверх проекта (заменит `BattleController`, `Projectile`; добавит `BattleBuffs`, `BattleBuffPicker`, editor ит.17).
2. Дождись компиляции.
3. В меню Unity: **BrickGame → Setup Iteration 17 (Battle Buffs)**

Скрипт строит пикер в сцене Battle и прокидывает ссылку в `BattleController`. Идемпотентен.

## 5. Как тестировать
1. Пройди демо-фазу → COLLECT → BATTLE.
2. В начале боя — оверлей «CHOOSE A BUFF» с 3 картами. Выбери одну.
3. Бой стартует с эффектом (например, TWIN SHOT — по два снаряда; BIG SHELLS — взрывы; SLOW FIELD — враги ползут).
4. Переиграй уровень — выпадет новый случайный выбор, прошлый буст не действует.

## 6. Известные нюансы / тюнинг
- Сила бустов аркадная (множители в `BattleController.BeginBattle` и радиус взрыва в `Projectile.explosionRadius`).
- TWIN SHOT тратит 1 заряд на 2 выстрела (по двум передним целям).
- Если врагов мало — мульти-цель/пробитие менее заметны; можно поднять `enemyCount` в `BattleController`.
- Пикер появляется до спавна врагов (без таймера) — спокойно выбираешь.

### Тех. детали
- Буст хранится в статике `BattleBuffs` и сбрасывается в `BattleController.Start` перед показом пикера.
- Пикер использует `.SetUpdate(true)` (на случай любых пауз) и нормальное время.
- `Projectile.explosive` использует `Physics2D.OverlapCircleAll` + урон всем `Enemy` в радиусе.
- Скрипты без namespace и без комментариев.
