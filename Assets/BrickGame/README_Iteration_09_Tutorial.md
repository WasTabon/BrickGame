# BrickGame — Iteration 09: Tutorial (без текста)

## 1. Что было добавлено
Туториал без единого слова: на 1-м уровне (и только до первого тапа) показывается подсветка-кольцо + «тап-рябь» у низа башни и стрелка в сторону ямы — подсказка «тяни нижние блоки, чтобы упало в яму». Первый тап скрывает подсказку и больше она не появляется.

## 2. Что изменилось с прошлой итерации
- Добавлен `Tutorial` (Game-сцена). Срабатывает только если `GameSession.Level == 1` и флаг `TutDone` ещё не выставлен.
- Ничего из прошлых итераций не тронуто; подсказки — мировые спрайты без коллайдеров, инпут не перехватывают.

## 3. Editor скрипты: что запустить
1. Скопируй папку `Assets/BrickGame` поверх проекта (добавит `Tutorial.cs` и текстуры кольца/стрелки).
2. Дождись компиляции.
3. В меню Unity: **BrickGame → Setup Iteration 9 (Tutorial)**

Скрипт генерит спрайты кольца и стрелки, добавляет объект `Tutorial` с тремя спрайтами и прокидывает ссылки. Идемпотентен.

> Полезно для теста: **BrickGame → Reset Tutorial Flag** сбрасывает `TutDone`, чтобы туториал показался снова.

## 4. Как тестировать
1. (Если уже проходил) выполни **BrickGame → Reset Tutorial Flag**.
2. Убедись, что `GameSession.Level == 1` (свежий запуск — по умолчанию 1).
3. Открой `Game.unity`, Play.
4. Должны появиться: пульсирующее кольцо + расходящаяся рябь у низа башни и стрелка, указывающая в сторону ямы.
5. Сделай первый тап — подсказки плавно исчезают и при следующих заходах не появляются.

## 5. Ожидаемый результат
- Кольцо мягко пульсирует, рябь циклично расходится и затухает у нижней части башни (точка «тяни сюда»).
- Стрелка покачивается в сторону ямы (куда должно упасть).
- Первый тап (по миру, не по UI) скрывает туториал и ставит флаг `TutDone=1`.
- На 2+ уровнях туториал не показывается.

## 6. Известные ограничения текущей итерации
- Подсказка статична по позиции (рассчитана под башню 1-го уровня); это ок, т.к. показывается только на уровне 1.
- Используется «тап-рябь» вместо рисованной руки (без текста и без сложного арта) — читается как «тапни здесь».
- Сброс флага — только через меню редактора (в билде туториал одноразовый на устройство).

## 7. Что будет в следующей итерации
Финальный проход (ит.10): haptics на ключевые действия, донастройка screen shake, мелкие анимации, проверка 60 FPS и общий вычитка/подчистка.

### Тех. детали
- `Tutorial` гейтится по `GameSession.Level==1` и `PlayerPrefs "TutDone"`.
- Спрайты-подсказки без коллайдеров — не мешают броскам; dismiss по первому тапу вне UI.
- Анимации на DOTween (пульс/рябь/покачивание), исчезновение — DOFade.
- Скрипты без namespace и без комментариев.
