# BrickGame — Iteration 18: VFX (рендер-независимые)

> Всё здесь работает на текущем Built-in рендере и **не сломается при замене спрайтов** — это VFX поверх любой графики. Замена арта (спрайтов) ничего не настраивает заново.

## 1. Что было добавлено
- **Снаряд/пушка:** trail на снаряде, muzzle flash + дымок из пушки, искры+вспышка при попадании.
- **Камера-фидбек:** лёгкий зум-пульс и hit-stop (freeze-frame) на крупных ударах/обрушениях.
- **Яма:** подсветка/пульс краёв, когда блок над ямой; рябь при сборе; мини-шкала заполнения в HUD.
- **Виньетка:** лёгкое затемнение краёв экрана (приближение «пост-вида» без URP).

## 2. Что НЕ вошло (нужен URP)
bloom, color grading, chromatic aberration, 2D-свет (свечение/rim-light/мягкие тени) требуют перевода проекта на **URP 2D** — это смена render pipeline (ручной шаг, может покрасить материалы). Когда переключишь проект на URP, я отдельной итерацией добавлю Volume-профиль (bloom/vignette/CA) и Light2D. Короткая инструкция — в конце файла.

## 3. Что изменилось (скрипты)
- `Vfx` (новый) — статик-хелпер: вспышки и искры (спавнит временные спрайты, сам удаляет).
- `MuzzleFlash` (новый) — на дуле пушки; вспышка+дым при выстреле.
- `CameraJuice` (новый) — зум-пульс камеры (на главной камере Game).
- `PitGlow` (новый) — подсветка краёв ямы, когда блок над ней.
- `Projectile` — trail (рантайм-материал) + вспышка/искры при попадании.
- `BattleController` — дёргает muzzle flash на выстреле.
- `SlowMoController` — добавлен `Freeze()` (hit-stop).
- `ImpactManager` — на сильный удар: hit-stop / slow-mo / зум камеры (взаимоисключающе по порогам).
- `PitJuice` — рябь при сборе.
- `GameHUDController` — мини-шкала заполнения (`SetProgress`).
- `LevelManager` — вешает `PitGlow`, передаёт спрайт ряби, обновляет шкалу.

## 4. Editor скрипты: что запустить
1. Скопируй папку `Assets/BrickGame` поверх проекта (много замен + новые VFX-скрипты + editor ит.18).
2. Дождись компиляции.
3. В меню Unity: **BrickGame → Setup Iteration 18 (VFX)**

Скрипт генерит спрайты искры/виньетки, вешает `CameraJuice` на камеру Game, `MuzzleFlash` на дуло (Battle), задаёт hit-спрайт снаряду, добавляет шкалу в HUD и виньетку во все сцены. Идемпотентен. `PitGlow`/рябь подключаются в рантайме (без правок сцены).

## 5. Как тестировать
1. Бой: выстрелы оставляют trail, у дула вспышка+дым, при попадании — искры; BIG SHELLS даёт крупную вспышку.
2. Демо-фаза: крупное обрушение → лёгкий зум/стоп-кадр.
3. Когда блок летит над ямой — края ямы подсвечиваются; при сборе — рябь; вверху растёт шкала заполнения.
4. По краям экрана — мягкое затемнение (виньетка).

## 6. Тюнинг
- Пороги камеры/стопа: `ImpactManager.cameraZoomThreshold/slowMoThreshold/freezeThreshold`.
- Сила зума: `CameraJuice.Punch` (0.06).
- Trail: параметры в `Projectile.Awake` (time/width/color).
- Виньетка: сила в генераторе (`*0.55`) — перегенерится при удалении `Textures/vignette.png` и повторном Setup.
- Подсветка ямы: `PitGlow` (скорость/пульс).

## 7. Как включить URP (для bloom/2D-света) — кратко
1. Package Manager → установить **Universal RP**.
2. Project → Create → Rendering → **URP 2D Renderer** (+ URP Asset).
3. Project Settings → Graphics → выставить URP Asset; в нём указать 2D Renderer.
4. Edit → Rendering → Materials → **Convert** существующих материалов в URP (важно, иначе «маджента»).
5. На камеру добавить **Volume** + профиль (Bloom/Vignette/Chromatic Aberration).
Когда сделаешь шаги 1-4 — пришли, и я добавлю профиль Volume и Light2D настройкой/скриптом.

### Тех. детали
- Все VFX — отдельные временные объекты или тинты/скейл существующих; спрайты можно менять свободно.
- Trail-материал создаётся в рантайме на шейдере `Sprites/Default` (Built-in).
- Скрипты без namespace и без комментариев.
